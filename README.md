## 3. train WGAN-GP
```

```
- Execute below command in GPU
```
python main.py --model WGAN-GP \
               --is_train True \
               --generator_iters 40000 \
               --cuda True \
               --batch_size 64
```
- Execute below command in order to check result
```
tensorboard --logdir logs/
```
